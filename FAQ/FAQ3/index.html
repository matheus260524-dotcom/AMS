python

Oque é python

é uma linguagem de programação de alto nível, interpretada e de código aberto. Famosa pela sua sintaxe simples que lembra a linguagem humana

O que é uma lista e qual a diferença entre ela e uma Tupla?

Listas e tuplas são estruturas de dados usadas para armazenar coleções de itens.

Como funcionam os comandos condicionais (if, elif, else)?

Eles permitem controlar o fluxo de execução do código com base em condições lógicas. O if testa a primeira condição, o elif testa condições alternativas caso o if seja falso, e o else captura todos os outros casos.

Qual é a finalidade do método input() e qual é o tipo de dado que ele retorna por padrão?

A finalidade do método input() é pausar a execução do programa e aguardar que o usuário digite uma informação pelo teclado. Por padrão, ele retorna todos os dados inseridos como uma string (texto), mesmo que o usuário digite apenas números

O que é uma variável e quais são as regras básicas para nomeá-las no Python?

Uma variável em Python funciona como um espaço na memória do computador usado para armazenar dados (como textos ou números).

JavaScript

Oque é JavaScript?

é uma linguagem de programação interpretada de alto nível, amplamente utilizada para tornar sites e aplicativos interativos e dinamicos

O que javascript pode alterar?

O JavaScript é uma linguagem de programação extremamente versátil que roda no navegador do usuário (client-side) ou no servidor (Node.js). A sua função principal é adicionar interatividade, permitindo alterar bastantequalquer aspecto de uma página web, dados de sistemas e até funcionalidades de dispositivos.

Quais são os principais tipos de dados primitivos que existem no JavaScript?

O JavaScript possui 7 tipos de dados primitivos. Eles são valores imutáveis e representam a forma mais simples de informação na linguagem.

O que é o conceito de DOM e como alteramos um texto na página com ele?

O DOM (Document Object Model) é uma interface que o navegador cria para representar o seu código HTML como uma árvore de objetos.

Como podemos declarar uma função tradicional e qual a vantagem das Arrow Functions?

As funções tradicionais são declaradas utilizando a palavra-chave function. As arrow functions oferecem vantagens como sintaxe mais concisa (especialmente para callbacks) e um escopo léxico fixo para o this, o que simplifica o código e evita erros de contexto.

c++

Oque é c++?

C++ é uma linguagem de programação de propósito geral e alto desempenho

O que o c++ consegue fazer?

O C++ é uma das linguagens de programação mais poderosas e versáteis do mundo. Poroferecer controle direto sobre o hardware e exigir alto desempenho, ele é uma base de softwares que desativa velocidade e eficiência máximas.

Qual é a diferença fundamental entre as linguagens C e C++?

A principal diferença fundamental entre C e C++ está no paradigma de programação. Enquanto o C é uma linguagem procedural (baseada em funções e estruturas de dados separadas), o C++ foi criado como uma extensão do C que adiciona suporte à Orientação a Objetos (uso de classes, herança e polimorfismo)

Como funcionam os ponteiros e para que serve o operador asterisco (*)?

Um ponteiro é uma variável especial que não armazena um valor direto (como um número), mas sim o endereço de memória onde esse valor está guardado. Em vez de trabalhar com uma cópia dos dados, o ponteiro permite acessar ou alterar a variável original diretamente no seu endereço na memória

java

Oque é java

O Java é uma das linguagens de programação e plataformas de software mais populares do mundo

Qual a principal diferença entre Classes e Objetos em Java?

A principal diferença é que a classe é o modelo ou a planta (um conceito abstrato), enquanto o objeto é a materialização desse modelo (uma entidade concreta)

O que acontece se você tentar dividir um número inteiro por zero em Java?

Em Java, tente dividir um número inteiro por zero interrompido a execução do programa e lança um erro crítico conhecido como ArithmeticExceptioncom a mensagem "/ by zero". Isso ocorre porque a divisão de inteiros por zero é uma operação indefinida na matemática

Por que o Java é considerado uma linguagem multiplataforma?

O Java é considerado uma linguagem multiplataforma graças ao princípio "escreva uma vez, execute em qualquer lugar" (WORA)

O que é o JVM (Java Virtual Machine) e qual a sua principal função?

A JVM (Java Virtual Machine) é o componente central da plataforma Java que atua como um motor de software para executar programas. Sua principal função é converter o código compilado (conhecido como bytecode) em instruções nativas que podem ser entendidas pelo sistema operacional e pelo hardware do usuário.

PHP

Oque é PHP

é uma linguagem de programação de código aberto voltada principalmente para o desenvolvimento web

Qual símbolo usamos para terminar uma linha de comando em PHP?

O símbolo usado para terminar uma linha de comando em PHP é o ponto e vírgula (;).

Como funciona o array associativo no PHP e como podemos percorrê-lo usando o laço foreach?

Um array associativo no PHP é uma estrutura de dados que mapeia valores para chaves personalizadas.

Como funciona a concatenação de textos usando o ponto (.) no PHP?

A concatenação em PHP é a junção de duas ou mais strings para formar uma única cadeia de caracteres. O operador utilizado para isso é o ponto (.). A nova string criada é o resultado da união exata dos operandos, da esquerda para a direita.

O que significa a sigla PHP e qual o seu papel principal no desenvolvimento web?

PHP significa PHP: Hypertext Preprocessor (um acrônimo recursivo). Seu papel principal é atuar como uma linguagem de programação executada no lado do servidor (backend), responsável por gerar páginas web dinâmicas, processar formulários, gerenciar sessões de usuários e interagir com bancos de dados.

Swift

Oque é Swift

é uma linguagem de programação moderna e orientada a objetos, desenvolvida pela Apple

Para que servir o comando print()em Swift?

O comando print() em Swift é utilizado para exibir mensagens no console durante a execução do programa.

É obrigatório usar ponto e vírgula ;no final das linhas em Swift?

Não, em Swift não é obrigatório usar ponto e vírgula no final das linhas.

Qual é a diferença prática entre criar uma variável com let e uma com var?

A diferença prática entre let e var reside no escopo (onde a variável pode ser acessada) e no comportamento de declaração.

Como funcionam as duas formas principais de desembrulhar uma Optional: if let e guard let?

As duas principais formas de desembrulhar uma Optional em Swift de forma segura são if let e guard let. Ambas evitam erros de tempo de execução ao verificar se a variável contém um valor antes de usá-la, mas possuem focos e escopos diferentes.

C#

Oque é C#

é uma linguagem de programação moderna e orientada a objetos, desenvolvida pela Microsoft

Para que servir uma estrutura ifno C#?

A estrutura if no C# é utilizada para executar um bloco de código apenas se uma condição específica for verdadeira.

Como você escreve um comentário de apenas uma linha no código?

Em C#, você escreve um comentário de apenas uma linha usando duas barras (//) antes do texto do comentário.

O que é o .NET e qual a sua relação com a linguagem C#?

O .NET é uma plataforma de desenvolvimento de código aberto e gratuita criada pela Microsoft. Ele fornece o ambiente e as ferramentas necessárias para criar, executar e compilar diversos tipos de softwares. Já o C# (C-Sharp) é uma linguagem de programação moderna e orientada a objetos, projetada especificamente para ser executada dentro da plataforma .NET.

Para que serve a palavra-chave var e como o compilador descobre o tipo da variável?

A palavra-chave var serve para declarar variáveis de forma implícita, permitindo que você omita o tipo de dado explícito no código (como int, String ou double). Ela torna o código mais limpo e legível, eliminando a repetição desnecessária de tipos, mantendo a segurança de um sistema fortemente tipado.

typescript

O que é o TypeScript e qual a principal diferença dele para o JavaScript tradicional?

O TypeScript é um superconjunto do JavaScript desenvolvido pela Microsoft. Na prática, ele funciona como uma "camada extra" sobre o JavaScript, adicionando principalmente tipagem estática

O que significa dizer que o TypeScript é uma linguagem fortemente tipada?

Dizer que o TypeScript é uma linguagem fortemente tipada significa que o seu sistema de tipos é rigoroso e que as variáveis possuem um tipo definido que deve ser respeitado durante todo o código. O compilador restringe operações inválidas e proíbe que tipos incompatíveis sejam misturados acidentalmente.

Qual é a diferença prática entre usar um type alias e uma interface?

Tanto type quanto interface servem para descrever o formato de dados no TypeScript, mas diferem em versatilidade, capacidade de expansão e comportamento prático.

Como funciona a inferência de tipos (Type Inference) quando criamos uma variável sem definir o tipo dela?

A inferência de tipos é a capacidade de uma linguagem de programação deduzir automaticamente o tipo de dado de uma variável com base no valor atribuído a ela.

Para que serve o arquivo tsconfig.json em um projeto TypeScript?

O arquivo tsconfig.json indica que o diretório é a raiz de um projeto TypeScript. Ele serve para indicar ao compilador quais arquivos incluir/excluir e definir as regras de compilação (como converter código TypeScript para JavaScript).
